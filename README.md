# Machine Learning Algorithms from Scratch

This repository contains implementations of classic machine learning algorithms written from scratch using only fundamental Python and numerical tools. The goal is to build a clear understanding of how these algorithms work under the hood.

> Currently implemented: **Linear Regression**
